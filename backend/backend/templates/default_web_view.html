Welcome to the Health Chatbot API! Use our API to engage in interactive conversations with a virtual Health agent. Below you'll find information on how to use the `/chatbot/` endpoint.

Endpoint: /chatbot/

This endpoint accepts POST requests to initiate or continue a conversation with the chatbot.

Request Format

Send a JSON object with the following structure:

{
  "chat_history": [
    {"type": "human", "content": "Hello! What can you tell me about HealthCare?", data_type:'la_medicaid'}
  ],
  "question": "Do i qualify for this service?"
}
