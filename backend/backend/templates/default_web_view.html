Mardi Gras Chatbot API

Welcome to the Mardi Gras Chatbot API! Use our API to engage in interactive conversations with a virtual Mardi Gras enthusiast. Below you'll find information on how to use the `/chatbot/post/` endpoint.

Endpoint: /chatbot/post/

This endpoint accepts POST requests to initiate or continue a conversation with the chatbot.

Request Format

Send a JSON object with the following structure:

{
  "chat_history": [
    {"type": "human", "content": "Hello! What can you tell me about Mardi Gras?"},
    {"type": "ai", "content": "Mardi Gras is a vibrant celebration with parades, masquerade balls, and lots of music and dancing."}
  ],
  "question": "Can you recommend some traditional foods?"
}

Response Format

The response will include the updated chat history and the latest response from the chatbot:

{
  "chat_history": [
    {"type": "human", "content": "Hello! What can you tell me about Mardi Gras?"},
    {"type": "ai", "content": "Mardi Gras is a vibrant celebration with parades, masquerade balls, and lots of music and dancing."},
    {"type": "human", "content": "Can you recommend some traditional foods?"},
    {"type": "ai", "content": "Certainly! Be sure to try King Cake, jambalaya, gumbo, and beignets."}
  ],
  "latest_response": "Certainly! Be sure to try King Cake, jambalaya, gumbo, and beignets."
}

Note: Replace the content of `chat_history` and `question` with your specific conversation context and query.
